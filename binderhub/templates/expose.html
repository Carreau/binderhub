Register with MyBinder.org

This page allow you to register the current binder instance with https://mybinder.org.

By click the following button you will have the option to select this binder as the default one to use when clicking on a binder link. It is necessary to click this button to tell mybinder.org that you are allowed to access this binder instance. Your default binder can be changed an anytime by heading to the mybinder.org setting page and selecting a new default.
